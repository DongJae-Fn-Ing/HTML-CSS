브라켓에서 주석 넣기

주석(comment) 단축키 : ctrl +/

블록(여러 줄) 주석 또는 현재 커서 위치의 주석 ctrl + shift + /

브라켓 사이드바 열고 닫기 : ctrl + alt + h

콘텐츠 이동 : ctrl + shift + 상하 화살표(상하)

컨텐츠 복사 : ctrl + d

동일한 태그 여러 개 작성: *

  • div>p 는 div의 자식요소 p가 만들어진다.

    아이디 속성: #아이디명

    태그명#아이디명

    클래스 속성:.클래스명

    태그명.클래스명

    일반 속성:[속성명=값]

    태그명[속성명=속성값]

    번호 붙이기: $

    $는 일련번호를 생성한다.

    3부터 시작하는 번호 붙이기 $@3

    시작값은 $@뒤에 숫자로 지정

    텍스트 콘텐츠 삽입:{텍스트}
